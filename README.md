# JobHunt AI Agent

AI-powered job matching and application tracking platform developed as a diploma project.

## Overview

JobHunt AI Agent is an intelligent job search assistance system that combines classical information retrieval, semantic matching, and AI-generated explainability to support job seekers in finding relevant opportunities and managing their application process.


---

## Technology Stack
### Frontend
HTML, CSS, JavaScript
### Backend
Python, FastAPI, SQLAlchemy, JWT Authentication, Pydantic
### Database
PostgreSQL
### AI / Matching
Scikit-learn, TF-IDF, Cosine Similarity, SBERT, OpenAI API
### Dynamic Dataset
Adzuna API
### Infrastructure
Docker, Docker Compose
### Tools
Git, GitHub
